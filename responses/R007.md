# Prompt
멀티탭에 멀티탭을 꽂아서 사용하고 있는데, 이거 별 상관 없는건가? 멀티탭에는 헤어드라이어, 에어컨, 고데기를 같이 사용하고 있어,

# Raw AI Response

## 결론: 지금 구성은 **빼는 게 맞아** ⚠️

**멀티탭에 멀티탭을 꽂고**, 거기에 **헤어드라이어 + 에어컨 + 고데기**를 같이 쓰는 건 **화재 위험이 꽤 높은 조합**이야.
특히 에어컨·드라이어는 소비전력이 큰 편이고, 고데기도 전열기구라서 멀티탭 과부하·접촉불량·발열 리스크가 커져.

소방청 자료에서도 냉방기기는 **벽면 단독 콘센트 사용**, **문어발식 멀티탭 금지**를 권고하고 있고, 광주 지역 냉방기기 화재 사례에서도 과부하·접촉불량 같은 전기적 요인이 주요 원인으로 언급됐어. ([소방청][1])
또 멀티탭은 정격 용량을 확인하고, 안전상 **허용 전력의 약 80% 이내**로 쓰는 게 바람직하다는 설명도 있어. ([소방청][2])

---

## 왜 위험하냐면

대충 계산하면 이래.

| 제품     |                 대략 소비전력 |
| ------ | ----------------------: |
| 헤어드라이어 |          약 1,200~2,000W |
| 에어컨    | 제품/운전상태 따라 수백~2,000W 이상 |
| 고데기    |            약 30~150W 내외 |

문제는 **합산 전력**이야.
예를 들어 드라이어 1,600W + 에어컨 1,200W만 해도 **2,800W**고, 여기에 고데기까지 붙으면 일반 멀티탭 기준으로 꽤 부담이 커져.

특히 멀티탭을 한 번 더 물리면:

* 접점이 늘어남
* 접촉불량 가능성 증가
* 전선/플러그 발열 증가
* 오래된 멀티탭이면 내부 접점 열화 가능성 증가
* 먼지·습기 있으면 합선 위험 증가

부산소방 실증실험에서도 10A 일반 멀티탭에 고소비전력 에어컨을 연결했을 때 과전류와 발열 문제가 확인됐고, 소비전력이 큰 제품은 벽면 콘센트에 직접 연결해야 한다고 강조했어. ([소방청][3])

## 지금 해야 할 조치

**1. 에어컨은 멀티탭에서 빼고 벽 콘센트에 단독 연결**
가장 중요해. 에어컨은 가능하면 **벽면 콘센트 하나를 혼자 쓰게** 해야 해.

**2. 드라이어와 고데기는 에어컨과 같은 멀티탭에서 동시에 쓰지 않기**
드라이어는 짧게 쓰는 기기지만 순간 부하가 커서 에어컨과 같이 쓰면 위험도가 확 올라가.

**3. 멀티탭-멀티탭 연결은 해제**
이건 편해서 자주 하게 되는데, 고출력 기기에서는 피해야 해.

**4. 멀티탭이 뜨겁거나 냄새나면 즉시 중단**
아래 증상 있으면 바로 빼는 게 맞아.

* 플러그나 멀티탭이 손대기 어려울 정도로 뜨거움
* 타는 냄새
* 지지직 소리
* 콘센트 주변 변색
* 플러그가 헐거움
* 차단기 자주 내려감

## 정리하면

**에어컨 + 드라이어 + 고데기 + 멀티탭 연쇄 연결 = 하지 않는 게 맞음.**
에어컨은 **벽 콘센트 단독**, 드라이어/고데기는 **다른 콘센트에서 필요할 때만 사용**하는 쪽으로 바꿔. 🔌

벽 콘센트가 부족하면, 에어컨 쪽은 임시 멀티탭으로 해결하기보다 **전용 콘센트 추가 시공**이 더 안전한 선택이야.
